# ⚙️ Methods

This folder contains Java programs that demonstrate the use of methods in Java. These programs were completed as part of my undergraduate laboratory exercises and helped me understand method creation, parameter passing, return values, method overloading, and code reusability.

## 📂 Programs Included

* Add Two Numbers
* Find Maximum Number
* Factorial Using Method
* Prime Number Using Method
* Fibonacci Series Using Method
* Calculator Using Methods
* Method Overloading

## 🛠️ Language Used

* Java

## 📚 Learning Outcomes

* Creating and calling methods
* Passing parameters to methods
* Returning values from methods
* Improving code reusability
* Understanding method overloading

More method-based programs will be added as I continue learning.
