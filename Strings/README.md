# 📝 Strings

This folder contains Java programs that demonstrate string manipulation and common string operations. These programs were completed as part of my undergraduate laboratory exercises to strengthen my understanding of Java's String class and text processing techniques.

## 📂 Programs Included

* Reverse String
* Palindrome String
* Count Vowels
* Count Words
* String Length
* String Comparison
* String Concatenation
* Character Frequency

## 🛠️ Language Used

* Java

## 📚 Learning Outcomes

* Understanding string handling in Java
* Using built-in String methods
* Performing string manipulation
* Solving text-based programming problems
* Improving logical thinking through string operations

More string-based programs will be added as I continue learning.
