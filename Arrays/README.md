# 📊 Arrays

This folder contains Java programs that demonstrate the use of one-dimensional and two-dimensional arrays. These programs were completed as part of my undergraduate laboratory exercises and helped me understand array operations, searching techniques, and matrix manipulation.

## 📂 Programs Included

* One-Dimensional Array
* Two-Dimensional Array
* Largest Element in an Array
* Smallest Element in an Array
* Sum of Array Elements
* Linear Search
* Binary Search
* Matrix Addition

## 🛠️ Language Used

* Java

## 📚 Learning Outcomes

* Understanding one-dimensional and two-dimensional arrays
* Reading and displaying array elements
* Performing array operations
* Implementing searching algorithms
* Working with matrices

More array-based programs will be added as I continue learning.
