# 🔁 Loops

This folder contains Java programs that demonstrate the use of looping statements. These programs were completed as part of my undergraduate laboratory exercises to understand iteration, control flow, and problem-solving using Java.

## 📂 Programs Included

* Factorial
* Fibonacci Series
* Prime Number
* Palindrome Number
* Reverse Number
* Sum of Digits

## 🛠️ Language Used

* Java

## 📚 Learning Outcomes

* Understanding `for`, `while`, and `do-while` loops
* Developing logical thinking through iterative programming
* Solving number-based problems using loops
* Improving problem-solving skills with Java

More loop-based programs will be added as I continue learning.
