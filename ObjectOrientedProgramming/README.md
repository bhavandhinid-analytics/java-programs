# ☕ Object-Oriented Programming (OOP)

This folder contains Java programs that demonstrate the fundamental concepts of Object-Oriented Programming (OOP). These programs were completed as part of my undergraduate laboratory exercises and helped me understand how Java uses classes and objects to build reusable, maintainable, and scalable applications.

## 📂 Programs Included

* Class and Object
* Constructor
* Inheritance
* Method Overriding
* Polymorphism
* Encapsulation
* Abstraction
* Interface

## 🛠️ Language Used

* Java

## 📚 Learning Outcomes

* Creating classes and objects
* Understanding constructors
* Applying inheritance to reuse code
* Implementing method overriding and polymorphism
* Protecting data using encapsulation
* Working with abstract classes and interfaces

More Object-Oriented Programming examples will be added as I continue learning.
